# 打卡

## 日常打卡

`GET https://bk.jd.com/m/money/home/daka.html`

#### 打卡成功

```json
{
    "resultCode": "0000",
    "resultMessage": "打卡成功",
    "data": null,
    "success": true
}
```

#### 打卡失败, 今日已领

```json
{
    "resultCode": "0001",
    "resultMessage": "已经打过卡了，每天只能打一次！",
    "data": null,
    "success": false
}
```

#### 打卡失败, 需先完成领钢镚任务

```json
{
    "resultCode": "0003",
    "resultMessage": "请先领取钢镚",
    "data": null,
    "success": false
}
```

## 领钢镚任务

`GET https://bk.jd.com/m/money/home/recDoJobMoney.html?pcId=82`

#### 领取成功

```json
{
    "resultCode": null,
    "resultMessage": null,
    "data": null,
    "success": true
}
```

#### 领取失败 (比如已经领了...)

```json
{
    "resultCode": "-1",
    "resultMessage": "没有找到当前奖品",
    "data": null,
    "success": false
}
```
